{% blocktrans trimmed %} After selecting “ADD CONTENT” on the course site, you can choose a type of the content which you want to create.

  • PDF
  • Text
  • LaTeX
  • YouTube
  • Bild
  • Panopto
  • Exercise
  • General URL
  • Anki Deck
Here we choose LaTeX content as an example: {% endblocktrans %}

  • {% blocktrans trimmed %} in ① you can click on the “?” to see detailed instruction for the latex code field. {% endblocktrans %}
  • {% blocktrans trimmed %} in ② you can select “ADD ATTACHMENT” to select an image file from your local computer as an attachment to this content. {% endblocktrans %}
  • {% blocktrans trimmed %} after you click on “ADD ATTACHMENT” there will also appear a “REMOVE ATTACHMENT” button, which allows you to remove the last attachment. {% endblocktrans %}
  • {% blocktrans trimmed %} in ③ you can determine wether or not other users can edit this content. {% endblocktrans %}
  • {% blocktrans trimmed %} in ④ you can mark a content as publicly visible. This means that the content will be visible to all users, even if they are not logged in.
    Note: This option is affects only courses that are publicly visible.
    Please keep in mind that it is not usefull to mark content as public if the content requires a login to be viewed, e.g. the mjority of Panopto videos.
