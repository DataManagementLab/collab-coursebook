  • {% blocktrans trimmed %} in ① you can select “ACTIONS” in order to access the following functions: {% endblocktrans %}
    • {% blocktrans trimmed %} “Approve / Disapprove”: If you are a moderator of a course, you can approve or disapprove content as audited. {% endblocktrans %}

    • {% blocktrans trimmed %} DUPLICATE COURSE: Duplicate this course {% endblocktrans %}

    • {% blocktrans trimmed %} DELETE COURSE: Delete the course {% endblocktrans %}
  • {% blocktrans trimmed %} in ② you can click on the name of a specific topic for quick topic jumping, and click on the hamburger button to toggle/untoggle the topic list. {% endblocktrans %}
  • {% blocktrans trimmed %} in the “Coursebook” part you can see marked coursebook content. In ③ you can also click on the hamburger button in order to toggle/untoggle the coursebook section. {% endblocktrans %}
